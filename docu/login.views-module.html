login_user(request)

source code 

Funcion de login de usuario.

Variables username = Nombre de usuario. password = Clave de usuario. state = Estado del login, puede ser logueado, esperando datos, o demostrar un error de datos incorrectos.

La funcion recibe en el request los datos de usuario determina si es correcta y esta activada. La funcion retorna por medio del context el estado del login y el username.
